ğŸ“– Functional Description
Add Product

Users can add a new product by entering the title, price, image URL, and category. Required field validation is applied.

Display Product

All products are displayed dynamically in a responsive card layout.

Edit Product

Users can edit any product. The selected product data is loaded into the form and can be updated.

Delete Product

Users can delete any product from the list. The UI updates instantly.

Sorting

Products can be sorted by price in ascending and descending order.

Searching

Users can search products by title using the search input.

Local Storage

Product data is saved in browser local storage so data is not lost after page refresh.
